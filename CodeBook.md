# Modifications
* Train and test data sets are combined to a sinngle data set.
* Measurements on the mean and standard deviation for each measurement are extracted.
* Attributes are labeled with descriptive names by using "features.txt" and "activity_labels.txt" which are in the original dataset.
* Now, the total dataset has been melted by using "melt" function and then "cast" function has been applied.
===========================================================================
